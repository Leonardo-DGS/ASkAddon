Types

Json Message
Pattern:
  • %jsonmessage%
  • json [message] [(of|from)] %string%
Since: 1.0
A json message that can be further customized, or can be sent to a player.
Examples:
json message of "I'm a json message"
Boss Bar
Pattern:
  • %bossbar%
  • new boss[ ]bar
Since: 1.0
A boss bar that can be customized and shown to players.
Examples:
new boss bar
