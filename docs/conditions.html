Conditions

String Starts With
Pattern:
  • %string% start[s][ with] %string%
  • %string% (doesn't|don't|not) start[s][ with] %string%
Since: 1.0
Check if a string starts with another string.
Examples:
player's name starts with "example string"
player's name doesn't starts with "example string"
String Ends With
Pattern:
  • %string% end[s][ with] %string%
  • %string% (doesn't|don't|not) start[s][ with] %string%
Since: 1.0
Check if a string ends with another string.
Examples:
player's name starts with "example text"
player's name doesn't starts with "example text"
Player Has Potion Effect
Pattern:
  • [player] %player% has [potion] effect [type] [of] %potioneffecttype%
  • [player] %player% has(n't| not) [potion] effect [type] [of] %potioneffecttype%
Since: 1.0
Check if a player has the potion effect specified.
Examples:
player has potion effect type night vision
player hasn't potion effect type night vision
File Exists
Pattern:
  • file %string% exist[s]
  • file %string% (not|doesn't|don't) exist[s]
Since: 1.0
Check whether a file exists or not.
Examples:
file "examplefile.txt" exists
file "examplefile.txt" doesn't exists
Exists Object In Ram
Pattern:
  • exists object [with name] %string% [saved] in ram
Since: 1.1.6
Check if exists an object, with specified name, previously saved in ram by the effect wich does that.
Examples:
exists object with name "exampleobjectname" in ram
