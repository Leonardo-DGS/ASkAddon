Effects

Send Title
Pattern:
  • send title %string% [with subtitle %string%] [for %timespan%] [with %timespan% fade in] [with %timespan% fade out] to %player%
Since: 1.0
Sends a title to a player.
Examples:
on player join:
send title "Hello" with subtitle "Welcome in my awesome server!" with 8 seconds fade in and 9 seconds fade out to player
Set Action Bar
Pattern:
  • set action[ ]bar of %player% to %string%
Since: 1.0
Set the action bar of a player to the string supplied
Examples:
on command /actionbar:
set action bar of player to "Wonderful action bar!"
Load World
Pattern:
  • load world %string%
Since: 1.0
Loads the world with name the string supplied.
Examples:
load world "Spawn"
Unload World
Pattern:
  • unload world %string%
Since: 1.0
Unloads the world with name the string supplied.
Examples:
unload world "Spawn"
Enable/Disable Keep Inventory
Pattern:
  • set keep inventory to %boolean%
Since: 1.0
Enable/Disable keep inventory for a player in death events.
Examples:
on death of player:
set keep inventory to true
Enable/Disable Keep XP
Pattern:
  • set keep (xp|level) to %boolean%
Since: 1.0
Enable/Disable keep XP for a player in death events.
Examples:
on death of player:
if player has permission "keep.xp":
set keep xp to true
else:
set keep level to false
Send Json Message
Pattern:
  • send %jsonmessage% to %player%
Since: 1.0
Sends a json message to a player.
See "Json Message" in page "Expressions" for more informations.
Examples:
command /jsonmessage:
send json of "I'm a simple message" tooltip "I'm a tooltip" suggest "/examplecommand" run "/anothercommand"
Show Boss Bar
Pattern:
  • show[ boss[ ]bar] %bossbar%
Since: 1.0
Shows a boss bar.
See "New Boss Bar" in page "Expressions" for more informations.
Examples:
command /bossbar [player]:
set {_bossbar} to new bossbar
add player player-argument to [boss][ ][bar] %bossbar%
show {_bossbar}
Hide Boss Bar
Pattern:
  • hide[ boss[ ]bar] %bossbar%
Since: 1.0
Hides a boss bar.
See "New Boss Bar" in page "Expressions" for more informations.
Examples:
command /bossbar [player]:
set {_bossbar} to new bossbar
add player player-argument to [boss][ ][bar] %bossbar%
hide {_bossbar}
Add Flag to Boss Bar
Pattern:
  • add flag %flag% to[ boss[ ]bar] %bossbar%
Since: 1.0
Adds a flag to a boss bar.
Examples:
command /bossbar [player]:
set {_bossbar} to new bossbar
add player player-argument to [boss][ ][bar] %bossbar%
show {_bossbar}
add flag create fog to {_bossbar}
Add Player to Boss Bar
Pattern:
  • add player %player% to [boss][ ][bar] %bossbar%
Since: 1.0
Adds the player specified to a boss bar.
Examples:
command /bossbar [player]:
set {_bossbar} to new bossbar
add player player-argument to [boss][ ][bar] %bossbar%

Create File
Pattern:
  • create[ (a|the)] file %string%
Since: 1.0
Creates a new file with the name supplied.
Examples:
create the file "playerdata.yml"
Delete File
Pattern:
  • delete[ the] file %string%
Since: 1.0
Deletes the file with name supplied.
Examples:
delete file "playerdata.yml"
Request Garbage Collector
Pattern:
  • [(request|do|make) [a]] garbage collect[or]
Since: 1.0
Request a garbage collector, for clear unused data in memory and free RAM memory.
Examples:
request a garbage collector
Create World
Pattern:
  • create [a] [new] world [(with name|called|named)] %string%
Since: 1.0
Creates a new world if not exists with the name supplied.
Examples:
create a new world with name "PvP"
Delete World
Pattern:
  • delete [the] world [(with name|called|named)] %string%
Since: 1.0
Deletes the world named as the string supplied.
Examples:
delete the world named "PvP"
Load Chunk
Pattern:
  • load chunk [at] %chunk%
Since: 1.0
Loads a chunk.
Examples:
loop loaded chunks of player's world:
load chunk loop-chunk
Unload Chunk
Pattern:
  • unload chunk [at] %chunk%
Since: 1.0
Unloads a chunk.
Examples:
loop loaded chunks of player's world:
unload chunk loop-chunk
Save Object In Ram
Pattern:
  • save object %object% in ram [with name] %string%
  • save object %object% [with name] %string% in ram
Since: 1.0
Save any type of variable in ram (instead of save it to variables file) for use it later.
Examples:
command /note [<text>]:
save object in ram with name "%player%.note"
message "note saved"
Delete Object In Ram
Pattern:
  • delete object [with name] %string% [saved] in ram
  • delete all object[s] [saved] in ram
Since: 1.0
Delete an object previously saved in ram or delete all objects previously saved in ram by this plugin.
Examples:
command /deletenote:
delete object with name "%player%.note" saved in ram
command /deleteallobjects:
delete all objects in ram
