Triggers

Player Change World
Pattern:
  • player change[d] world
Since: 1.0
Called when a player goes in another world.
Examples:
on player change world
on player changed world
Player Change Main Hand
Pattern:
  • player change[d] main hand
Since: 1.0
Called when a player changes their main hand in the client settings.
Examples:
on player change main hand
on player changed main hand
Player Swap Hand Items
Pattern:
  • player swap hand [items]
Since: 1.0
Called when a player swap items between main hand and off hand using the hotkey.
Examples:
on player swap hand
on player swap hand items
Player Chat Tab Complete
Pattern:
  • player [chat] tab complete
Since: 1.0
Called when a player attempts to tab-complete a chat message.
Examples:
player tab complete
player chat tab complete
Player Edit Book
Pattern:
  • [player] (edit book|book edit)
Since: 1.0
Called when a player edits or signs a book and quill item. If the event is cancelled, no changes are made to the BookMeta.
Examples:
on player edit book
on player book edit
on book edit
on edit book
Player Shear Entity
Pattern:
  • player shear entity
Since: 1.0
Called when a player shears an entity.
Examples:
on player shear entity
Player Change Velocity
Pattern:
  • player [change] velocity
Since: 1.0
Called when the velocity of a player changes.
Examples:
player velocity
player change velocity
Player Armor Stand Manipulate
Pattern:
  • player armor stand manipulate
Since: 1.0
Called when a player interacts with an armor stand and will either swap, retrieve or place an item.
Examples:
on player armor stand manipulate
