ASkAddon Documentation

ASkAddon is an addon for the plugin Skript for Bukkit/Spigot, it adds more effects, expressions, events and conditions

This is ASkAddon documentation. You will find all supported features of ASkAddon plugin here.

In Skript's syntax expressions the text in square brackets is optional eg: [text], and text like this (text|othertext) means that you have to write only one of the strings in round brackets separated by character '|'. If you find something of incorrect in this documentation report it to issue tracker.
